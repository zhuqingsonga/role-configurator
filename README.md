# OpenClaw 角色配置助手 (role-configurator)

帮你轻松完成 OpenClaw 的角色配置，是每个新用户的第一个 skill。

## 功能特点

- **21 个预设角色模板**，覆盖生活、商业、营销、母婴、情感等全场景
- **引导式配置**，用对话的方式完成角色配置
- **一键生成配置文件**，自动写入 SOUL.md
- **智能推荐 skill**，配置完成后推荐相关 skill

## 文件结构

```
openclaw-role-configurator/
├── SKILL.md              # 技能说明文档
├── README.md             # 本文件
├── _meta.json            # 元数据
├── data/
│   ├── role_templates.json       # 21个角色模板
│   └── skill_recommendations.json # skill 推荐规则
└── scripts/              # 核心脚本
    ├── template_manager.py       # 角色模板管理器
    ├── config_writer.py          # 配置文件写入器
    └── skill_recommender.py      # skill 推荐器
```
